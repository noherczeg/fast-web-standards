# Fast Web

A simple Proof of Concept project showcasing up-to-date standards in networking and web-development.

- [X] Docker
- [X] nginx
- [ ] HTTPS
- [ ] HTTP2
- [ ] ES Modules
- [ ] Lazy imports
- [ ] Lazy CSS imports

## Build
```
./build.sh
```

## Run
```
./run.sh
```

## Debugging
```
docker exec -it ${CONTAINER_ID} /bin/sh
```
